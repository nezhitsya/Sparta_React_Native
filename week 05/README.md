# 앱개발 종합반 - 5 주차

### [수업 목표]
1. 수익형 앱
2. 구글 광고, 애드몹 적용
3. 배포

## 01. 5 주차
구글 광고 플랫폼을 이용한 수익 창출

## 02. 수익 창출 방식
- 앱 마켓에 유료앱 배포 수익 모델
- 앱 내 배너 광고 수익 모델
   - 배너 클릭
   - 배너 광고 시청
   - 배너 광고 사용자 참여
- 앱 콘텐츠 판매 수익 모델 (인 앱 결제)
- 구독 수익 모델
- 앱 개발 용역 수익 모델
- 외부 브랜드 광고 수익 모델

## 03. 애드몹 (AdMob) - 설정
AdMob : 앱 내 구글 배너광고를 쉽게 붙일 수 있으며 수익 현황 알림 및 일정 수익 이상 시 환전 가능
